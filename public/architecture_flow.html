01

표현 계층 (Presentation Layer)

사용자가 보고 만지는 모든 것. React와 Tailwind CSS를 통해 빠르고 아름다운 화면을 구현합니다.

02

비즈니스 로직 계층 (Logic Layer)

웹사이트의 두뇌 역할. Server Actions와 Middleware가 사용자의 요청을 검증하고 데이터를 가공합니다.

03

데이터 계층 (Data Layer)

정보의 고향. PostgreSQL에 데이터가 안전하게 저장되며, RLS 보안 정책으로 허가된 사람만 접근할 수 있게 지킵니다.

기술 스택 상세 (Technology Detail)

Frontend
Next.js 15+

최신 웹 표준 기술로, 서버에서 화면을 미리 그려 보내는 SSR 기능을 지원합니다.

Tailwind CSS

유틸리티 우선 방식의 디자인 도구로, 일관성 있는 미니멀 UI를 완성합니다.

Backend
Server Actions

API 서버 없이도 서버 코드를 함수처럼 호출하여 브라우저 네트워크 노출을 줄입니다.

Supabase Auth

복잡한 회원가입/비밀번호 찾기/로그인 상태 유지를 클라우드로 해결합니다.

Database
PostgreSQL

가장 신뢰받는 관계형 데이터베이스로, 게시글과 관계 정보를 체계적으로 관리합니다.

Row Level Security

데이터베이스 자체에서 보안을 검토하여, 해킹 위협으로부터 정보를 보호합니다.
